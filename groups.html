  • 移动组

    加入移动组，你将通过企业级项目学习到移动端和网站开发的技术，你将收获和一群志同道合的战友共同攻克项目的成就感和革命友谊。移动组是个有极客氛围和创造力的项目组。 累计开发过的产品有：白云黄鹤移动端（华科官方论坛）、稻草、VPoker、非常购等..... 与我们合作过公司有：腾讯、弗洛格信息科技、绿荫者联盟等..

    UI设计,PM,Web,Android,IOS

  • 迅测组

    加入迅测组，你将有机会参与完整的软硬件产品开发流程，和资深的前华为工程师以及前武汉中岩科技总工程师一起工作的机会。在这里，你可以通过实践学习嵌入式开发、服务器后台开发以及PC软件开发等技术。迅测组开发的电动车智能防盗产品“小安宝”已投入市场，“钻孔电视采集系统”和孔径测试仪曾斩获大奖。目前在研项目有“桥梁绕度动态采集系统”和“船舶通风管道消音系统”

    技术方向：

    嵌入式,服务器后台,PC软件,移动开发

  • 微弹幕组

    我们组的产品微弹幕是一款线下活动的互动工具，解决了线下活动中主办方与观众之间的现场实时互动需求。产品已经服务了全国近45万人次、4万场活动，累计毛收入超过6万人民币，使用范围已经覆盖了全国各省，平均用户增长率稳定在每季度50%。我们拥有2份软件著作权、1个商标，2个发明专利正在审核中，在同类市场上具有一定的技术壁垒、时间壁垒、用户壁垒和品牌壁垒。

    技术方向：

    全栈

  • 挪威组

    挪威组与Confirmit公司(挪威)合作，是一个对互联网数据进行挖掘的项目组。这些数据来源于世界各地的论坛以及博客，使用xquery对网页上的信息进行提取和分析。除此之外，也会负责一些微信开发，web开发等方面的项目。在项目过程中，你有机会接触到web前后台开发，Linux平台的常规使用，热门的脚本语言Python

    技术方向：

    Java,Python,Html,Xquery,XML

  • 魅果科技组

    魅果创业组，依托于创业师兄的公司，提供真真切切的项目体验。“百万级用户活跃量，上万行服务器代码”，安卓IOS大神细心为你指导代码，CTO全栈工程师亲自为你规划项目。客户端开发，lua,java,cocos2dx一样不落;服务器维护，python,js,php样样精通。大公司流程，敏捷开发周周新功能; 小公司氛围，插科打浑人人享福利。

    技术方向：

    客户端：lua + java + cocos2d 服务器：python + js + html

  • 华三组

    华三组是Dian团队与杭州华三公司合作进行真实交换机/路由器软件项目开发的项目组。加入我们，你将与来自惠普/华三的经验丰富的软件工程师一起工作、交流，掌握业界流行的软件敏捷开发流程，学习熟悉Linux环境下编程，在实践中学习计算机网络的知识。目前，华三组正进行着两个与智能存储区域网络（SmartSAN）有关项目，SmartSAN是大数据、云计算时代数据中心网络发展的趋势，在华三组，你将始终与网络通信的前沿接轨

    技术方向：

    网络通信,Linux C/C++,多进程,多线程

  • 华为组

    Dian团队华为组与华为武汉研究所合作，在华为公司资深员工的指导下，进行软件项目的研究与开发，其成果将会用到华为公司的生产和产品中。目前，华为组正进行着智能路径生成、畸变图像矫正、AP测速系统、主备数据同步等四个不同方向的问题。在华为组，你将获得亲身参与世界500强公司项目开发的机会，同时也能从算法到工程实践全方位提升自己

    技术方向：

    机器学习,图像处理,Android,Linux C/C++
